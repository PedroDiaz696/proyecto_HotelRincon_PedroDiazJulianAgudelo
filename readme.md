# REQUISITOS DEL PROYECTO

1. Sitio web sencillo, agradable, funcional que atraiga visualmente a los clientes y lo invite a revisar y apartar habitaciones del hotel

2. Movile first (responsive)

3. ## debe tener 3 paginas:
    - Pagina principal (landing page)

    -debe contener elementos carrusel de habitaciones, areas de hotel, muestras fotograficas de servicios de comida, espa, comida y zonas humedas-

    - Consultar disponibilidad y gestionar reservas

    -debe poder consultar fechas de interes, cuantas personas se van a hospedar y el sistema arrojara las habitaciones disponibles que esten de acuerdo a los filtros con el valor total correspondiente a noches y personas que tomaran el servicio-

    - Pagina con la ubicacion, direccion y diferentes formas de contactar


# FUNCIONALIDADES DE LA PAGINA
    1. solo los usuarios registrados en el sitio web podran hacer reservas o cancelacion de disponibilidad por lo cual deben registrarse con el numero de identificacion
